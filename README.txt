This repository is an e-commerce made with React-JS.
To try it you need: 
1. clone the repository, you can do it clicking on the green box " <> Code".
2. allocate yourself in the terminal within the directory of the repository.
3. write in the terminal: npm run dev.
4. it generate the local host that will be available.
5. open the local host url in your navegator of prefference (google, zafari, ms internet, mozilla)
6. you'll can interact with the webpage by clicking in the specie you want to see, then choose the image that you want, can be one or more.
9. click on the cartwidget to see the items you choosed, quantity and total price, put your name in the name space then click on "realizar compra".
10. your order will be saved in the orders database on Firebase.
